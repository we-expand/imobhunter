### 1. **Proxycurl** - Dados do LinkedIn (MELHOR OPÇÃO)
- 🌐 **Site**: https://nubela.co/proxycurl/
- 💰 **Preço**: 
  - 🎁 10 créditos GRÁTIS
  - $0.01-0.03 por perfil
  - Pay-as-you-go (paga apenas o que usar)
- ✅ **Por que escolher?**
  - Dados PÚBLICOS do LinkedIn (100% legal)
  - Perfis completos com experiência + educação
  - Company data + lista de funcionários
  - Email finder integrado
  - Rate limit: 300 req/min
- 📖 **Como configurar**:
  1. Acesse https://nubela.co/proxycurl/
  2. Clique "Sign Up" (pode usar Google)
  3. Dashboard → API Keys → Generate New Key
  4. Cole a key no sistema e clique "Testar"

4. Role até **"Assistente de Configuração de API"**
5. Escolha um provedor (recomendo Apollo ou Proxycurl)
6. Siga o passo a passo visual
7. Cole a API key e clique "Testar"

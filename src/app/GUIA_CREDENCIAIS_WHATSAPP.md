## 🎫 PARTE 3: OBTER AS 3 CREDENCIAIS

**Agora você está na página: "WhatsApp → API Setup"**

### 📍 CREDENCIAL 1: TOKEN DE ACESSO

#### LOCALIZAÇÃO:

**No topo da página, procure seção:**

```
╔══════════════════════════════════════════════╗
║  Etapa 1: Selecionar números de telefone     ║
║                                              ║
║  De:                                         ║
║  +351 912 345 678  [Gerenciar números]       ║
║  Phone number ID: 123456789012345            ║
║                                              ║
║  ──────────────────────────────────          ║
║                                              ║
║  Para:                                       ║
║  +1 555 025 5555                            ║
║                                              ║
╠══════════════════════════════════════════════╣
║  Etapa 2: Enviar mensagens com a API        ║
║                                              ║
║  📝 Configuração temporária                  ║
║     Você pode testar imediatamente usando    ║
║     este token de acesso e número de         ║
║     telefone, que expiram em XX dias.        ║
║                                              ║
║  Token de acesso:                            ║
║  EAAABsbCS1iHgBO3kZBfxB4hR8ZAM... [Copiar]  ║
║                                              ║
╚══════════════════════════════════════════════╝
```

#### COMO COPIAR:

1. **Encontre a seção:** "📝 Configuração temporária" ou "Temporary access token"

2. **Veja o token:**
   - Começa com: `EAAA...`
   - É uma string LONGA (50-300 caracteres)

3. **Copie o token:**
   - **Método A:** Clique no botão **"Copiar"** ao lado
   - **Método B:** Selecione todo o texto e Ctrl+C (Cmd+C no Mac)

4. **Cole num arquivo de texto:**
   - Abra Bloco de Notas / TextEdit
   - Cole (Ctrl+V ou Cmd+V)
   - Salve como: `credenciais-whatsapp.txt`

**Exemplo do que você copiou:**
```
TOKEN:
EAAABsbCS1iHgBO3kZBfxB4hR8ZAMN37uZCwcyUcOZCEXFJiVV4SZAGPnD...
```

✅ **CREDENCIAL 1 COPIADA!**

### 📍 CREDENCIAL 2: PHONE NUMBER ID

#### LOCALIZAÇÃO:

**Na mesma página, mais para cima.**

Procure a seção: **"De:"** ou **"From:"**

```
╔══════════════════════════════════════════════╗
║  De:                                         ║
║  ──                                          ║
║  +351 912 345 678  [Gerenciar números]       ║
║                                              ║
║  Phone number ID: 123456789012345  ← ESTE!  ║
║                    ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑            ║
╚══════════════════════════════════════════════╝
```

#### COMO COPIAR:

1. **Encontre:** "Phone number ID:"

2. **Selecione APENAS os números:**
   - Exemplo: `123456789012345`
   - São só números (sem espaços, sem letras)

3. **Copie:**
   - Selecione os números
   - Ctrl+C (ou Cmd+C)

4. **Cole no arquivo de texto:**
   ```
   TOKEN:
   EAAABsbCS1iHgBO3kZBfxB4hR8ZAM...
   
   PHONE_NUMBER_ID:
   123456789012345
   ```

✅ **CREDENCIAL 2 COPIADA!**

### 📍 CREDENCIAL 3: BUSINESS ACCOUNT ID

**Esta é a mais difícil de encontrar. Vou mostrar 3 métodos:**

---

#### MÉTODO 1: Via URL (MAIS FÁCIL)

1. **Abra uma nova aba no navegador**

2. **Cole este link:**
   ```
   https://business.facebook.com/wa/manage/home/
   ```

3. **Você chegará no WhatsApp Manager**

4. **Olhe a URL na barra de endereços do navegador:**

   Você verá algo tipo:
   ```
   https://business.facebook.com/wa/manage/home/?waba_id=123456789012345&...
   ```

5. **Copie APENAS o número depois de `waba_id=`**

   Exemplo:
   ```
   waba_id=123456789012345
            ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
            COPIE ISTO
   ```

6. **Cole no arquivo:**
   ```
   TOKEN:
   EAAABsbCS1iHgBO3kZBfxB4hR8ZAM...
   
   PHONE_NUMBER_ID:
   123456789012345
   
   BUSINESS_ACCOUNT_ID:
   123456789012345
   ```

✅ **CREDENCIAL 3 COPIADA!**

---

#### MÉTODO 2: Via Configurações (ALTERNATIVO)

1. **No WhatsApp Manager:**
   ```
   https://business.facebook.com/wa/manage/home/
   ```

2. **Menu lateral esquerdo:**
   - Clique: **"Configurações"** ou **"Settings"**

3. **Na primeira linha verá:**
   ```
   WhatsApp Business Account ID: 123456789012345
   ```

4. **Copie o número**

---

#### MÉTODO 3: Via Meta Business Settings (ALTERNATIVO)

1. **Acesse:**
   ```
   https://business.facebook.com/settings/whatsapp-business-accounts
   ```

2. **Você verá lista de contas WhatsApp**

3. **Clique no nome da sua conta:** "AI LeadGen Pro"

4. **Abrirá painel lateral à direita**

5. **Verá:** "ID: 123456789012345"

6. **Copie o ID**

## 🆘 RESOLUÇÃO DE PROBLEMAS

### ❌ "Não vejo botão Criar App"

**Solução:**
- Certifique-se que está em: developers.facebook.com
- Faça login
- Procure no canto superior direito
- Ou clique ícone de 9 quadradinhos

### ❌ "Não vejo produto WhatsApp"

**Solução:**
- Role a página toda para baixo
- Ou use Ctrl+F e busque "whatsapp"
- Pode estar na segunda ou terceira linha de produtos

### ❌ "Token não aparece"

**Solução:**
- Certifique-se que conectou WhatsApp ao app
- Vá em: WhatsApp → API Setup (menu lateral)
- Procure "Configuração temporária"
- Se não vê, atualize a página (F5)

### ❌ "npm test dá erro"

**Causa:** Credenciais incorretas

**Solução:**
1. Abra .env
2. Verifique se copiou credenciais completas
3. Sem espaços antes/depois do =
4. Sem aspas
5. Token começa com EAAA
6. IDs são só números

### ❌ "Invalid OAuth access token"

**Causa:** Token expirou ou inválido

**Solução:**
1. Volte em developers.facebook.com
2. Seu app → WhatsApp → API Setup
3. Copie novo token
4. Substitua no .env
5. Teste novamente

## 🎯 APIs GRATUITAS Recomendadas

### 1. Hunter.io ⭐️⭐️⭐️⭐️⭐️
**Por que usar:** Melhor para verificar se emails são válidos

#### ✅ Plano Gratuito
- **25 verificações de email/mês**
- **50 buscas de domínio/mês**
- Sem cartão de crédito necessário
- Para sempre grátis

#### 📝 Como Obter
1. Acesse: https://hunter.io
2. Clique em "Sign Up Free"
3. Preencha: Nome, Email, Senha
4. Confirme seu email
5. Acesse: https://hunter.io/api-keys
6. Copie sua API key (começa com algo como `a1b2c3d4...`)

#### ⚙️ Configurar no Supabase
```bash
Project Settings > Edge Functions > Secrets
Nome: HUNTER_API_KEY
Valor: sua_key_aqui
```

#### 💡 Dica Pro
- Use apenas para leads importantes (limite de 25/mês)
- Foque em verificar emails corporativos (@empresa.com)
- Emails pessoais (@gmail.com) raramente precisam verificação

---

### 2. Apollo.io ⭐️⭐️⭐️⭐️⭐️
**Por que usar:** MELHOR custo-benefício! Dados profissionais completos

#### ✅ Plano Gratuito
- **60 créditos/mês** (renova todo mês!)
- Dados de LinkedIn, telefones, emails
- Informações de empresa
- Sem cartão de crédito

#### 📝 Como Obter
1. Acesse: https://app.apollo.io/sign-up
2. Clique em "Sign Up Free"
3. Preencha: Nome, Email, Senha, Empresa
4. **IMPORTANTE:** Escolha "Sales/Business Development" como função
5. Pule o onboarding (pode fechar os modais)
6. Vá para: Settings > API & Integrations
7. Ou direto: https://app.apollo.io/#/settings/integrations
8. Clique em "Create API Key"
9. Copie sua key (começa com algo como `ak_...`)

#### ⚙️ Configurar no Supabase
```bash
Nome: APOLLO_API_KEY
Valor: sua_key_aqui
```

#### 💡 Dica Pro
- **ESTA É A MELHOR API!** Use sempre que possível
- 60 créditos = 60 pessoas enriquecidas/mês
- Dados mais atualizados que outras APIs
- Telefones costumam ser válidos

## 🔧 Como Configurar Cada Key no Supabase

### Passo 1: Acessar Supabase Dashboard
```
https://supabase.com/dashboard/project/SEU_PROJETO_ID
```

### Passo 2: Ir para Edge Functions Secrets
```
Project Settings (ícone de engrenagem) 
> Edge Functions (menu lateral)
> Secrets (tab)
```

### Passo 3: Adicionar Cada Secret
Clique em "New Secret" e adicione:

```
Nome: HUNTER_API_KEY
Valor: sua_hunter_key_aqui
```

```
Nome: APOLLO_API_KEY
Valor: sua_apollo_key_aqui
```

```
Nome: PDL_API_KEY
Valor: sua_pdl_key_aqui
```

```
Nome: FULLCONTACT_API_KEY
Valor: sua_fullcontact_key_aqui
```

### Passo 4: Testar
1. Admin Dashboard
2. Clique em qualquer usuário
3. Aguarde 10-30 segundos
4. Veja dados sendo coletados!

## 🆘 Problemas Comuns

### "API key inválida"
- ✅ Verificar se copiou corretamente
- ✅ Remover espaços antes/depois
- ✅ Verificar se key ainda é válida no painel da API

### "Limite excedido"
- ✅ Aguardar renovação mensal
- ✅ Ou fazer upgrade para plano pago
- ✅ Usar outras APIs enquanto espera

### "Dados não aparecem"
- ✅ Verificar logs do servidor
- ✅ Algumas APIs podem estar indisponíveis
- ✅ Normal ter 40-60% de sucesso (algumas pessoas não têm presença online)

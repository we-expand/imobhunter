## 🎯 **COMO USAR:**

### **1. Configure os Filtros:**
```
1. Digite Nome/Sobrenome (opcional)
2. Digite Cargo (ex: "CEO", "Marketing Manager")
3. Digite Empresa (ex: "Microsoft")
4. Selecione Indústria
5. Digite Localização
6. ⭐ NOVO: Clique nas redes sociais desejadas
7. Configure Match Score mínimo
```

### **2. Clique em "Iniciar Busca":**
```
- Loading animado aparece
- Busca em múltiplas APIs
- Resultados aparecem em grid
```

### **3. Interaja com os Resultados:**
```
✅ Selecione leads (checkbox)
📋 Copie email/telefone (botão copy)
🔗 Acesse redes sociais (botões coloridos)
📨 Envie email direto
➕ Adicione ao pipeline
📥 Exporte selecionados para CSV
```

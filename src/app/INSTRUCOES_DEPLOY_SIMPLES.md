# 🚀 COMO FAZER O DEPLOY - SUPER SIMPLES

## ✅ MÉTODO RECOMENDADO (Sem Terminal)

### 📌 **Passo 1: Acesse o Dashboard**
Abra este link no navegador:
```
https://supabase.com/dashboard/project/evdyqlrssgsktctjruuq/functions
```

### 📌 **Passo 2: Baixe o Projeto**
- No Figma Make, procure por "Export", "Download" ou "Download Project"
- Baixe o projeto completo para seu computador
- Você terá uma pasta com todos os arquivos

### 📌 **Passo 3: Abra o Arquivo do Servidor**
- Vá até a pasta baixada
- Navegue até: `supabase/functions/server/`
- Abra o arquivo `index.tsx` em qualquer editor de texto (Notepad, TextEdit, VS Code, etc)

### 📌 **Passo 4: Copie TODO o Código**
- Selecione TUDO (Ctrl+A no Windows, Cmd+A no Mac)
- Copie (Ctrl+C ou Cmd+C)

### 📌 **Passo 5: Cole no Dashboard**
- Volte para o Dashboard do Supabase (link do Passo 1)
- Clique na função "server" (ou em "New Function" se não existir)
- Cole o código copiado (Ctrl+V ou Cmd+V)
- Clique em "Deploy"

### 📌 **Passo 6: Aguarde**
- Aguarde 30-60 segundos
- Quando aparecer "Deployed successfully" = PRONTO! ✅

### 📌 **Passo 7: Teste**
- Recarregue sua aplicação ImobHunter (Ctrl+Shift+R)
- Faça uma busca
- Agora com DADOS REAIS! 🎉

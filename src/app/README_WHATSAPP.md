# 📱 WhatsApp Business API - REAL

## ⚡ Quick Start

```bash
cd backend-whatsapp
npm install
npm run start:simple
```

Depois acesse o frontend e clique em **"Conectar WhatsApp"**.

---

## 🎯 O que acontece:

1. **Backend inicia** em `http://localhost:3001`
2. **Frontend conecta** automaticamente
3. **QR Code REAL** é gerado pela biblioteca oficial do WhatsApp
4. **Você escaneia** com seu celular
5. **Conectado!** Mensagens reais enviadas

## 🐛 Erro?

### "Erro ao conectar"
→ Backend não está rodando  
→ Execute: `npm run start:simple`

### "Port 3001 in use"
→ Outra aplicação usando a porta  
→ Mate o processo ou mude a porta

### QR Code não aparece
→ Aguarde 10-15 segundos (primeira vez)  
→ Puppeteer está baixando Chromium

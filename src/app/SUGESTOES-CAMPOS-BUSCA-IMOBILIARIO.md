# 🎯 SUGESTÕES: Campos Ideais para Busca de Leads Imobiliários

## 📊 ANÁLISE DOS CAMPOS ATUAIS

### ❌ **PROBLEMAS IDENTIFICADOS:**

Os campos atuais são **GENÉRICOS para redes sociais**, não específicos para o mercado imobiliário:

**Campos Atuais:**
- ✓ Platform (Instagram, Facebook, Twitter)
- ✓ Content Type (posts, stories, reels, videos)
- ✓ Hashtags
- ✓ Mentions (@)
- ✓ Keywords
- ✓ Exclude Keywords
- ✓ Location (genérico)
- ✓ Date Range
- ✓ Engagement Level (high, medium, low)
- ✓ Min Likes/Comments/Shares
- ✓ Has Media
- ✓ Verified

### 🔴 **O QUE ESTÁ FALTANDO:**

1. **Intenção de Compra/Venda** - O usuário quer comprar, vender, alugar ou investir?
2. **Tipo de Imóvel Específico** - Casa, apartamento, terreno, comercial, cobertura?
3. **Faixa de Preço** - Budget mínimo e máximo
4. **Características do Imóvel** - Quartos, metragem, garagem
5. **Localização Específica** - Cidade, bairro, região, CEP
6. **Perfil do Lead** - Primeira casa, investidor, família, solteiro
7. **Urgência** - Compra urgente, vendedor motivado?
8. **Palavras-chave Imobiliárias** - Financiamento, FGTS, permuta, etc.

## 📞 CONCLUSÃO

Os campos atuais são **adequados para busca genérica em redes sociais**, mas **insuficientes para captar leads imobiliários qualificados**.

A implementação dos campos sugeridos acima aumentará:
- ✅ **Precisão** na identificação de leads
- ✅ **Qualificação** automática via IA
- ✅ **Taxa de conversão** (abordagem personalizada)
- ✅ **ROI** do sistema

**Recomendação:** Criar uma **aba específica "Busca Imobiliária"** com esses campos otimizados, mantendo a busca genérica como opção secundária.

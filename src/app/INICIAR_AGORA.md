## 🆘 PROBLEMAS COMUNS:

### ❌ "npm: command not found"
**Solução:** Instale Node.js
1. Acesse: https://nodejs.org/
2. Baixe a versão LTS (recomendada)
3. Instale
4. Feche e abra o Terminal
5. Tente novamente

### ❌ "Port 3001 already in use"
**Solução:** Porta ocupada
```bash
lsof -ti:3001 | xargs kill -9
npm run start:simple
```

### ❌ "Cannot find module"
**Solução:** Dependências não instaladas
```bash
cd backend-whatsapp
npm install
npm run start:simple
```

### ❌ QR Code não aparece (backend rodando)
**Solução:** Aguarde 10-15 segundos
- Primeira vez demora (baixa Chromium ~200MB)
- Seja paciente

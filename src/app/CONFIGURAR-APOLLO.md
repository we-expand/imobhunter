## 📋 Como Configurar (3 minutos)

### **Passo 1: Criar Conta Gratuita**

1. Acesse: https://app.apollo.io/#/signup
2. Preencha:
   - Email profissional
   - Nome completo
   - Senha
3. Ou use **Sign up with Google** (mais rápido)
4. Confirme o email (verifique sua caixa de entrada)

---

### **Passo 2: Completar Perfil**

Após confirmar email, você verá um wizard:
1. **What's your role?** → Selecione "Sales" ou "Marketing"
2. **Company size?** → Selecione o tamanho da sua empresa
3. **What are you looking to do?** → "Find new leads"
4. Clique em **"Continue"**

⚠️ **Importante**: Preencha com dados reais. Apollo pode bloquear contas com informações falsas.

---

### **Passo 3: Obter API Key**

1. Após login, clique no seu **avatar** (canto superior direito)
2. Clique em **"Settings"**
3. No menu lateral esquerdo, clique em **"API"** ou **"API Keys"**
4. Clique no botão **"Create API Key"**
5. Dê um nome: `"AI-Leads-System"`
6. **Copie a API Key** (algo como):
   ```
   xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   
⚠️ **ATENÇÃO**: Guarde esta chave em local seguro! Ela dá acesso aos seus créditos.

---

### **Passo 4: Configurar no Sistema**

1. Volte para o sistema → **Integrações** → **Apollo.io**
2. Vá para a aba **"Configuração"**
3. Cole a API Key no campo **"API Key do Apollo.io"**
4. Clique em **"Salvar Configuração"**
5. Clique em **"Testar Conexão"**

✅ Se aparecer "Conexão com Apollo.io estabelecida!" → **Sucesso!**

## ⚡ Integração com o Sistema

### **Importar Leads**
1. Faça a busca no Apollo
2. Marque os leads que quer importar (checkbox)
3. Clique em **"Importar Selecionados"**
4. Leads aparecem automaticamente no **Pipeline**!

### **Detecção Automática de Cluster**
O sistema detecta automaticamente o melhor cluster:
- C-Suite, Director → **High-End/Executivos**
- HR, People → **Parcerias/Relocation**
- Investor, Real Estate → **Investidores**
- Outros → **1ª Habitação**

### **Score Inicial**
Leads importados ganham score baseado em:
- Tem email? +20 pontos
- Seniority alta? +15-20 pontos
- Tem telefone? +15 pontos
- **Score inicial**: 30-75 (Cold Lead)

✅ **Nunca compartilhe** sua API Key

- Ou faça upgrade para plano pago (US$49/mês = 1000 créditos)

## ❓ Troubleshooting

### **Erro: "Invalid API Key"**
➜ Certifique-se que copiou a chave completa (sem espaços)

### **Erro: "Rate limit exceeded"**
➜ Aguarde 1 minuto. Apollo limita requests por minuto

### **Poucos resultados na busca**
➜ Relaxe os filtros (menos job titles, localizações mais amplas)

### **Créditos não atualizando**
➜ Clique em "Testar Conexão" para recarregar info

## ✅ PASSO 1: Apollo.io (Obrigatório para buscas)

### 📱 Criar Conta (2 minutos)
```
1. Abra: https://app.apollo.io/
2. Clique em "Sign Up"
3. Preencha:
   - Email: seu@email.com
   - Senha: (crie uma forte)
4. Confirme o email
```

### 🔑 Gerar API Key (2 minutos)
```
1. Login no Apollo
2. Clique no seu avatar (canto superior direito)
3. Clique em "Settings"
4. No menu lateral: "API" ou "Integrations"
5. Clique em "Create New API Key"
6. Nome: "LeadGen Pro"
7. COPIE A KEY (exemplo: ap-1234567890abcdef...)
   ⚠️ GUARDE BEM! Só aparece uma vez.
```

### ⚙️ Configurar no Supabase (1 minuto)
```
1. Abra: https://supabase.com/dashboard
2. Selecione seu projeto
3. Menu lateral → Settings → Edge Functions
4. Clique em "Environment Variables" ou "Secrets"
5. Adicionar:
   Name: APOLLO_API_KEY
   Value: (cole sua key aqui)
6. Clique em "Save" ou "Add"
7. Aguarde 30 segundos
```

### ✅ Testar (1 minuto)
```
1. Abra a plataforma
2. Vá em: Buscar Leads → Busca Avançada
3. Digite:
   - Empresa: "Microsoft"
   - Cargo: "Sales Director"
   - Localização: "Lisboa"
4. Clique em "Buscar"
5. ✅ Se aparecer leads reais = FUNCIONOU! 🎉
6. ❌ Se aparecer "dados DEMO" = Verifique a key
```

## 🆘 Problemas Comuns

### "Apollo retorna erro 401"
**Solução:**
1. Verifique se copiou a key completa
2. Confirme que está em "APOLLO_API_KEY" no Supabase
3. Aguarde 30 segundos após salvar
4. Recarregue a plataforma (F5)

### "Aparecem só dados DEMO"
**Solução:**
1. Apollo não está configurado
2. Siga PASSO 1 acima
3. A API key precisa estar correta

### "Buscas não retornam nada"
**Solução:**
1. Verifique se tem créditos no Apollo (Dashboard)
2. Tente busca mais ampla (ex: só "Sales Director" sem empresa)
3. Mude localização para país maior (ex: "United States")

## 🎓 Como Usar

### Navegação Básica
```
1. Acesse Configurações → RGPD
2. Veja o score de conformidade
3. Explore as 5 tabs
4. Use ações rápidas na sidebar
```

### Exercer Direito RGPD
```
1. Tab "Direitos dos Titulares"
2. Escolha o direito (ex: Acesso)
3. Clique em "Exportar Dados"
4. Preencha o formulário
5. Aguarde resposta (30 dias)
```

### Ver Auditoria
```
1. Tab "Auditoria"
2. Veja KPIs em tempo real
3. Explore timeline de logs
4. Filtre por tipo/data
5. Exporte relatório
```

### Consultar APIs
```
1. Tab "APIs & Transferências"
2. Veja lista de 9 APIs
3. Verifique localização
4. Confira nível de risco
5. Leia sobre garantias
```

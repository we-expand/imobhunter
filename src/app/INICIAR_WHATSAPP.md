# ⚡ INICIAR WHATSAPP - 2 COMANDOS

## 🎯 PARA FUNCIONAR:

### 1️⃣ Abra o terminal na pasta backend-whatsapp

```bash
cd backend-whatsapp
```

### 2️⃣ Execute (PRIMEIRA VEZ):

```bash
npm install
```

Aguarde 2-3 minutos.

### 3️⃣ Execute (SEMPRE):

```bash
npm run start:simple
```

**PRONTO!** Backend rodando.

## 📱 AGORA NO NAVEGADOR:

1. Acesse: `http://localhost:5173`
2. Vá em: **Configurações** → **Integrações**
3. Seção: **WhatsApp Business**
4. Clique: **"Conectar WhatsApp"** (botão verde grande)
5. QR Code REAL aparece
6. Escaneie com seu WhatsApp
7. Pronto! ✅

# 🔄 Como Deletar Usuário e Recriar do Zero

## ✅ PASSO A PASSO

### **1. Acesse a Aba "Segurança" 🛡️**
```
Dashboard → Clique em "Segurança" (ícone de escudo)
```

### **2. Role até "Zona de Perigo" ⚠️**
```
Final da página → Seção vermelha "⚠️ Zona de Perigo"
```

### **3. Escolha a Ação**

#### **OPÇÃO 1: Deletar Apenas Usuário** (Recomendado para seu caso)
```
┌─────────────────────────────────────────┐
│ Deletar Dados de Autenticação          │
├─────────────────────────────────────────┤
│ Remove apenas usuário e 2FA.           │
│ Mantém leads e atividades.             │
│                                         │
│ [Deletar Usuário]                      │
└─────────────────────────────────────────┘

✅ O QUE SERÁ DELETADO:
• Seu usuário (email/senha)
• Configurações de 2FA
• Sessões ativas

❌ O QUE SERÁ MANTIDO:
• Todos os leads
• Todas as atividades
• Todas as configurações do sistema
• Todas as integrações
```

**→ Clique em "Deletar Usuário"**  
**→ Confirme no popup**  
**→ Sistema te redireciona para tela de login**  
**→ Crie novo usuário normalmente!**

---

#### **OPÇÃO 2: Deletar TUDO** (Limpar sistema completo)
```
┌─────────────────────────────────────────┐
│ 🚨 Deletar TUDO                         │
├─────────────────────────────────────────┤
│ Remove TODOS os dados:                  │
│ usuário, leads, atividades, configs.   │
│ AÇÃO IRREVERSÍVEL!                     │
│                                         │
│ [Deletar Todos os Dados]               │
└─────────────────────────────────────────┘

🚨 O QUE SERÁ DELETADO:
• TUDO! (sistema completamente limpo)

⚠️ CONFIRMAÇÃO DUPLA:
1. Primeiro popup: confirmar
2. Digite "DELETAR TUDO" (maiúsculas exatas)
```

**→ Clique em "Deletar Todos os Dados"**  
**→ Confirme no primeiro popup**  
**→ Digite: `DELETAR TUDO` (exatamente assim)**  
**→ Sistema limpa tudo e redireciona para login**  
**→ Comece do zero!**

## 🎯 QUAL USAR?

### **Use "Deletar Usuário" se:**
✅ Quer testar login novamente  
✅ Quer recriar 2FA do zero  
✅ **Quer manter os dados de leads/atividades**  
✅ Só quer "resetar" seu acesso  

**→ É o seu caso! Use esta opção.**

---

### **Use "Deletar TUDO" se:**
✅ Quer limpar sistema completamente  
✅ Quer começar MVP do zero  
✅ Dados atuais são apenas de teste  
✅ Não precisa de nada que está salvo  

**→ Só use se realmente quer perder tudo!**

## 📝 EXEMPLO PRÁTICO

### **Cenário: Você quer testar login novamente**

**1. Vá em Segurança**
```
Dashboard → Segurança (última aba)
```

**2. Role até final da página**
```
Procure seção vermelha "⚠️ Zona de Perigo"
```

**3. Clique "Deletar Usuário"**
```
Primeiro botão na zona de perigo
```

**4. Confirme**
```
Popup: "⚠️ ATENÇÃO! Você irá deletar..."
Clique em "OK"
```

**5. Aguarde redirecionamento**
```
✅ Dados de autenticação deletados!
Redirecionando para login em 1 segundo...
```

**6. Crie novo usuário**
```
Tela de cadastro aparece
Preencha com:
• Nome
• Email (pode ser o mesmo de antes!)
• Senha
• Confirmar senha

Clique em "Criar Conta"
```

**7. Configure 2FA novamente**
```
Sistema gera novo QR Code
Escaneie com Google Authenticator
Valide código
Pronto!
```

## 🔐 SEGURANÇA

### **Confirmações:**
- **Deletar Usuário:** 1 confirmação
- **Deletar TUDO:** 2 confirmações + digitar texto

### **Não é possível desfazer!**
- Dados deletados são **permanentemente removidos**
- Não há backup automático
- Se precisar dos dados, exporte antes!

### **Como exportar antes (se precisar):**
```
1. Vá em "Dados"
2. Clique em "Exportar JSON"
3. Salva arquivo no computador
4. Depois pode importar novamente
```

## ❓ FAQ

### **P: Vou perder meus leads?**
**R:** Não, se usar "Deletar Usuário". Sim, se usar "Deletar TUDO".

### **P: Posso usar o mesmo email?**
**R:** Sim! Após deletar, pode criar novo usuário com mesmo email.

### **P: E se eu clicar sem querer?**
**R:** Tem confirmação. Leia bem antes de confirmar!

### **P: Quanto tempo demora?**
**R:** Instantâneo. Em 1 segundo está feito.

### **P: Preciso fazer logout antes?**
**R:** Não. O sistema faz logout automático.

### **P: E os dados no MongoDB?**
**R:** Se configurado, também apaga do MongoDB.
